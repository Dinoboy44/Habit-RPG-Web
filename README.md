# Habit RPG Web

A gamified habit tracking web application built using FastAPI and SQLite.

## Features
- User Registration & Login
- Session Authentication
- Add Habits
- Complete Habits
- Delete Habits
- XP & Level System
- SQLite Database
- Responsive Modern UI

## Tech Stack
- Python
- FastAPI
- SQLAlchemy
- SQLite
- Jinja2
- HTML
- CSS

## Installation

pip install -r requirements.txt

uvicorn main:app --reload
